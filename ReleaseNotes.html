LithoGen.py
Release Notes

Release Date: December 31, 2024

Features

  • Use as an input an image and create an STL file for the output.

Known Issues

  • Latency when generating an STL file especially for high-resolution images.
  • Cannot directly use the STL file as it needs tweaking to be able to slice it with a 3D printing software.
  • Only support JPG/JPEG image formats.

Additional Information

As an example for tweaking the 3D file, use blender :

  • Drag and drop the STL file into blender.
  • Select the object.
  • Use the "Remesh" tool and change the voxel size as needed.
